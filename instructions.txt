Git Basics Assignement - Your Instructions

1- Create a new folder and initialize the repository 
	git init
2- Paste the "instructions.txt" file into  this folder

3- Add a .txt fie named "file-1" containing any text of your choice to the working
directory.
	file-1.txt => this is our first assignement

4- Create a second .txt file named "file-2"
	 file-2.txt =>still working  on the assignement!

5- Add "file-1" and "file-2" to the staging area - don't add "instructions.txt"
	git add file-1.txt file-2.txt (not git add .)

6- change the initial text you added to "file-1"
	file-1.txt => add Just updated this text.

7- Now add all working directory files to the staging area
	git add .

8- Create the first commit
	git commit -m "added initial .txt files"

9- Create a second branch named "feature" (two commands are possible)
	git checkout -b feature / git switch -c feature
	
10-Add a third .txt file ("file-3.txt") to this branch
	file-3.txt => I was created in the faeture branch

11- create a new commit
	git add .  git commit -m "added file-3.txt"

12- Add the following text to "file-3" : "I will be deleted"
	file-3.txt => ... i will be deleted

13- Add the updated file to the staging area
	git add file-3.txt

14- Undo the staged change
	git reset file-3.txt / git restore --staged file-3.txt
	git checkout file-3.txt

15- Add the following text : "PLease add me to the master/main branch"
	file-3.txt => ... Please add me to the master/ main branch

16- Commit this latest change
	git add . git commit -m "added text to file-3.txt"

17- Merge the "master" (or "main") branch with "feature"
	git switch master  git merge  feature

18- Delete the "feature" branch
	git branch -D feature 

And most importantly : Have fun with the assignement :)
